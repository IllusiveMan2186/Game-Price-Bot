# Game Price Bot

## Docker Setup:

### To run GPB in Docker after installing, create a ".env" file and use the following command in root of project:

```console
docker-compose --env-file .env up
```

Environments in ".env" file :
* POSTGRES_DB_USERNAME;
* POSTGRES_DB_PASSWORD;
* GAMEZEY_LOGIN;
* GAMEZEY_PASSWORD;
* MAIL_USERNAME;
* MAIL_PASSWORD;
